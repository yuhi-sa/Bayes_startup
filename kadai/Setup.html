1. Homebrew

Homebrewとは

Wikipedia
Mac OS X オペレーティングシステム上でソフトウェアの導入を単純化するパッケージ管理システムのひとつ。

OS関連のパッケージをインストール、アンインストールできる管理システムのこと。

  2. スクリプトをコピーし、ターミナルで実行
  3. インストール確認

2. pyenv

pyenvとは

GitHub
pyenv lets you easily switch between multiple versions of Python. It's simple, unobtrusive, and follows the UNIX tradition of single-purpose tools that do one thing well.

Pythonのバージョンを管理できるコマンドラインツール。Python2系とPython3系を共存させることができる。

インストール方法

  1. Homebrewでpyenvをインストール。
  2. インストール確認。
  3. pyenvにPathを通す。
  4. 設定（Path）の適用。
# pyenvをインストール
$ brew install pyenv
 
# インストール確認
$ pyenv --version
# ログ
pyenv 1.2.17
 
# pyenvにPathを通す
$ echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.bash_profile
$ echo 'export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.bash_profile
$ echo 'eval "$(pyenv init -)"' >> ~/.bash_profile
 
# 設定の適用
$ source .bash_profile

3. Anaconda

Anacondaとは

Wikipedia
データサイエンスおよび機械学習関連アプリケーション（大規模データ処理、予測分析（英語版）、科学計算）のためのPythonおよびRプログラミング言語のフリーでオープンソースのディストリビューションであり、パッケージ管理および展開の単純化を目指したものである。

AnacondaをインストールすればPythonの標準的な環境を構築できる。

インストール方法

  1. pyenvでインストール可能なAnacondaを検索。
  2. Anacondaをインストール。
# インストール可能なAnacondaを検索
$ pyenv install -l | grep anaconda
 
# ログ
anaconda-1.4.0
︙
anaconda2-5.0.0
︙
anaconda3-5.3.1
 
# 3系の最新版をインストール
$ pyenv install anaconda3-5.3.1
 
# 切り替えられるをバージョン確認
$ pyenv versions
 
# ログ （*が現在指定しているバージョン）
  system
  anaconda2-5.3.1
  anaconda3-4.4.0
* anaconda3-5.3.1
 
# バージョン切り替え
# globalとすると全体に、localにするとそのカレントディレクトリ以下に指定したバージョンが反映される
$ pyenv global anaconda3-5.3.1
$ python --version
Python 3.7.0
 
$ pyenv local anaconda2-5.3.1
$ python --version
Python 2.7.16

4. MacTeX

MacTeXとは

TEX Wiki
MacTeX は TeX Live をベースにした macOS 専用の TeX ディストリビューション

tlmgrはTeXLiveのパッケージ管理ツールで、コマンドラインで使用する。

インストール方法

# MacTeXのインストール
$ brew cask install mactex
 
# アップデート
$ sudo tlmgr update --self --all
 
# ドキュメントサイズの設定
$ sudo tlmgr paper a4
 
# 日本語向けパッケージのインストール
$ sudo tlmgr install collection-langjapanese

5. Atom

Atomとは

Wikipedia
Atom（アトム）は、GitHubが開発したオープンソースのテキストエディタである。

PythonやTexのコーディングに使うとよい。テキストエディタは何を用いてもよいが小林先生のおすすめはこれ。

インストール方法

  1. Atomからファイルをインストール
  2. 展開し指示に従ってインストール

パッケージの追加方法

Atomを起動しツールバーからAtom > Preferences > Settings > installへ移動する。検索欄から好きなパッケージやテーマ（デザイン）をインストールできる。以下のパッケージをインストールしておくと良い。

基本パッケージ

  • atom-ide-ui : atomをideとして使用する際の機能をまとめた公式パッケージ
  • show-ideographic-space : 全角スペースに編集記号を表示（半角と全角の判別用）
  • project-manager : プロジェクトフォルダを管理する
  • minimap : minimapを表示する
  • minimap-cursorline : minimap上で今いるlineを表示する
  • minimap-find-and-replace : minimap 上で検索・置換単語を表示する
  • minimap-highlight-selected : minimap上で選択した単語をハイライトする
  • autocomplete-paths : 相対パスを入力補完する
  • file-icons : ファイルの色とアイコンを変えておしゃれにする
  • atom-beauty : 字下げ（インデント）などを自動でそろえてくれる
  • highlight-selected : 選択した単語をハイライトする

無効化推奨デフォルトパッケージ

  • spell-check : 綴間違いなどを教えてくれるパッケージだが日本語非対応

Python用パッケージ

  • atom-runner : Pythonなどのコードを実行する際に必要（Ctrl + Rで実行、Escか右上の×ボタンで閉じる事が可能）
  • autocomplete-python : Pythonコードを補完してくれる

適当なデレクトリにサンプルコードを作成し、実行（Ctrl + R）できればOK!

# sample.py 
import numpy as np
 
x = np.array([1, 2, 3])
print(x.dtype)

Tex用パッケージ

  • latex : AtomからLaTeXの文書スクリプトをコンパイルするのに必要 設定からEngineの項目をplatexする。PDF Producerの項目は以下の設定がおすすめ（Atomのpdfビュアーを使うため）。

    ☑ Move Result to Source Directory
    ☑ Build on Save
    □ Open Result after successful Build
    □ Open Result in Background

    Command + S でスクリプトのセーブ+ビルド+pdfのセーブを実行できる。

  • language-latex : Atom内でLaTeXのシンタックスハイライトのために必要
  • latexer : LaTeXコードの自動補完をしてくれる
  • pdf-view : AtomのためのPDFビューアー

適当なディレクトリにサンプルコードを作成、ビルド（Ctrl + S）できればOK!

% sample.tex
\documentclass{jarticle}
\usepackage{amsmath}
 
\begin{document}
 
\section{Test}
\begin{align}
  \sin(\theta)
\end{align}
 
\end{document}

ファイルの種類（シンタックス）は自動で判別され、シンタックスハイライトをつけてくれるが、うまく動作しない時は Ctrl + Shift + L でファイルの種類を手動で設定すればよい。
